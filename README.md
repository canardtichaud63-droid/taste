# TASTE Demo Frontend

Ceci est la version frontend de TASTE, votre concierge culturel.

## Instructions pour GitHub Pages

1. Créez un dépôt sur GitHub et nommez-le `taste-demo`.
2. Uploadez tous les fichiers et dossiers (`index.html`, `fonts/`, `README.md`).
3. Allez dans **Settings → Pages**, choisissez la branche `main` et enregistrez.
